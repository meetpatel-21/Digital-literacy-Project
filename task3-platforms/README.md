## Project Report: Skills Development & Assessment Tools

For this project, I utilised HackerRank as my primary platform for developing technical coding skills. I practised a variety of challenges focused on data structures, algorithms, and Python programming, which allowed me to move beyond theoretical knowledge and apply logic to solve real-world problems in a structured environment.

Additionally, I created an interactive assessment tool using Google Forms, linked to a Google Sheet. This setup enabled me to automate data collection and track results in real-time. I built a quiz that tests knowledge and provides immediate feedback, with all responses systematically organised in the spreadsheet for easy analysis.

These tools will benefit me academically. HackerRank sharpens my problem-solving skills, essential for technical coursework and exams. The Google Forms and Sheets integration will streamline my projects by providing a robust method for conducting surveys, managing group data, and automating simple grading tasks, enhancing both efficiency and accuracy.
